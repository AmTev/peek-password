# Peek Password Extension

A simple Chrome/Edge extension that adds password visibility toggle and copy functionality to any password field. Buttons appear automatically when you hover near password inputs.

## Features

- 👁️ **Secure Password Peek**: Hold to view password, automatically hides on release
- 📋 **One-Click Copy**: Instantly copy passwords to clipboard
- 🔄 **Site-Level Control**: Enable/disable functionality per website
- 🔒 **Zero Storage**: No password storage or data collection
- 🎯 **Minimal UI**: Clean, unobtrusive buttons
- 🌐 **Universal Support**: Works on all websites
- 🎨 **Modern Design**: SVG icons and smooth animations
- ⚡ **Performance**: Lightweight with minimal DOM impact
- 🔍 **Smart Detection**: Automatically finds password fields, even in dynamic content

## Installation

### Chrome Web Store
[Install from Chrome Web Store](https://chromewebstore.google.com/detail/peek-password/jcoinhahiiighiijhjoplndenopbfbfd)

### Manual Installation

1. Clone or download this repository
2. Navigate to `chrome://extensions` (Chrome) or `edge://extensions` (Edge)
3. Enable "Developer mode" in the top-right corner
4. Click "Load unpacked" and select the extension directory
5. Verify the extension icon appears in your browser toolbar

## Usage

1. Enable the extension through the toolbar icon
2. Hover near any password field to reveal:
   - 👁️ Eye button: Hold to view password, releases automatically
   - 📋 Copy button: Click to copy to clipboard with confirmation
3. Use the popup menu to:
   - Toggle extension globally
   - Manage per-site permissions
   - View and manage whitelisted sites
   - Remove individual sites from whitelist
   - Clear all whitelisted sites at once
